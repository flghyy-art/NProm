### 核心规则（别跳过）

- ⚠️ 每个 prompt 必须包含**裸露词** + **姿势词**
- ⚠️ 设备与画质必须匹配（手机 ≠ 8K，监控 ≠ masterpiece）
- ⚠️ `panties showing` + `pussy visible` = 冲突 → 用 `no panties, pussy visible`
- ⚠️ 禁止 `sheer/see-through/transparent` → 用 `unbuttoned/lifted/slipping off`
- ⚠️ 液体词必须用最小量词：`single drop, thin streak, faint trace`
- ⚠️ 纹身必带皮肤融合词（六词核心），漏带 = 贴纸感

## 🎯 推荐模型

**上下文长度是关键。** 本模板库内容丰富，完整读取需要大量 token。上下文不足的模型会截断内容、丢失规则，导致生成质量大幅下降。

| 推荐等级 | 模型                                 | 上下文 | 说明                                                                  |
| :------: | ------------------------------------ | ------ | --------------------------------------------------------------------- |
|  ⭐⭐⭐  | **DeepSeek V4 Pro / V4 Flash** | 1M     | 🏆 首选推荐。极强理解力+精准规则遵循，内容限制宽松，NSFW 生成质量最高 |
|  ⭐⭐⭐  | 小米 MiMo-V2.5 / MiMo-V2.5 Pro       | 1M     | 中文理解极好，内容限制宽松，Pro版推理更强，适合 NSFW 生成             |
|  ⭐⭐⭐  | Claude Opus 4.6 / Sonnet 4.6         | 1M     | 最佳理解力，精准遵循组合规则，内容限制少                              |
|  ⭐⭐⭐  | Gemini 3 Pro / 3 Flash               | 1M     | 超长上下文，适合一次读取全部模板，NSFW 限制比旧版更宽松               |
|   ⭐⭐   | GPT-5X                               | 1M     | 内容审查严格，NSFW 可能被拒或删减。需配合越狱 prompt                  |
|    ❌    | MiniMax M2X                          | 200K   | 理解力不足，复杂组合规则容易出错，不推荐                              |
|    ❌    | 开源小模型（<70B）                   | <128K  | 上下文短、规则遵循差、容易变形，不推荐                                |

**核心原则：上下文 ≥ 128K 且内容限制宽松的模型优先。DeepSeek V4 系列与小米 MiMo-V2.5 系列为当前最佳选择。**
