### 核心规则（别跳过）

- ⚠️ 每个 prompt 必须包含**裸露词** + **姿势词**
- ⚠️ 设备与画质必须匹配（手机 ≠ 8K，监控 ≠ masterpiece）
- ⚠️ `panties showing` + `pussy visible` = 冲突 → 用 `no panties, pussy visible`
- ⚠️ 禁止 `sheer/see-through/transparent` → 用 `unbuttoned/lifted/slipping off`
- ⚠️ 液体词必须用最小量词：`single drop, thin streak, faint trace`
- ⚠️ `09-负面提示词.md` 仅在 SD/ComfyUI 模型下生效

## 📖 简介

这是一套经过系统化整理的 AI 绘画提示词模板库，专注于 **NSFW（Not Safe For Work）** 领域的 prompt engineering。

风格方向：**亚洲面孔 / 日本 AV 审美 / 写实摄影风格**

模板设计目标：

- 🎯 **精准控制** — 从画质到姿势、从光影到液体，每个维度都有对应词库
- 🧬 **真实质感** — 通过瑕疵系统（imperfection system）避免 AI 生成的"塑料感"
- 🎲 **随机组合** — 每个模块都有随机抽取规则，确保每次生成都有变化
- 🚫 **避坑指南** — 标注了常见 AI 渲染错误和对应的负面提示词
- 🤖 **AI Agent 友好** — 附带装配公式表和风格配方，AI 可直接读取组装

## 🧩 模块说明

### 01 — 画质妆容 💎

画质标签按拍摄设备分类（专业相机 / 手机 / 监控 / 胶片 / 摄像头 / 无人机），自动匹配对应画质风格。妆容系统覆盖眼妆、唇妆、腮红、指甲，按场景（清纯 / 魅惑 / 高潮 / 哥特）提供搭配方案。

### 02 — 体型瑕疵 🧬

真实感瑕疵库，涵盖乳头/乳晕变异、面部不对称、身体纹路、拍摄瑕疵等。核心理念是**瑕疵增美感**——痣、雀斑、枕痕、湿发让画面"活"起来，而非堆砌缺陷。

### 03 — 裸露服装 👙

裸露分级词库（半裸 → 全裸 → 特写），包含女性/男性生殖器的精确解剖描述（内外阴唇、阴蒂、龟头等），以及液体细节系统（精液/淫水/母乳/尿液/汗液/唾液/泡沫）。每个液体类型都有量词、质地、光线、物理行为的精确控制。

### 04 — 姿势动作 🧘

200+ 姿势词库，按坐姿/躺姿/跪姿/站姿/性爱体位分类。含口交/深喉专项强化、阴茎×口腔/阴道空间关系精确描述、多人互动受力关系。

### 05 — 场景主题 🏞️

专门面向日本AV的场景分类库。内含 **60 个场景大类**（情人旅馆、电车痴汉、便利店、漫画喫茶、AV摄影棚、マジックミラー号、风俗店、和室、神社、监禁密室等）+ **30 个JAV经典主题**（人妻/OL/教师/护士/女仆/熟女/痴汉/NTR/凌辱/紧急/百合/面接/中出し/ぶっかけ等）+ 景别/视角/拍摄设备全套词库。不含任何科幻/奇幻/西方历史/非亚洲异域内容。

### 06 — 光影氛围 💡

12 种专业布光技法（发丝光/伦勃朗光/蝴蝶光/环形光等）+ 电影感布光 + 情色专用光影 + 场景×光影匹配速查表。

### 07 — 镜头风格胶片 🎞️

焦段分类（广角/标准/长焦/微距）、构图方式（三分法/对称/引导线等）、15 种胶片模拟（富士/柯达/理光 GR）、9 大电影镜头厂商（安琴/阿莱/库克/蔡司/Panavision/徕卡/适马/佳能/尼康）。

### 08 — 范例 ✨

77 例完整 prompt，覆盖校园/家居/职场/交通/购物/运动/户外/节日/特殊/胶片风格 10 个类别，展示各模块组合方式。

### 09 — 负面提示词 🚫

通用基础负面词 + 10 大场景专项负面词（手指/面部/液体/衣物穿透等），脚本可自动检测场景类型并匹配。

### 10 — 纹身标记 🏷️

50+ 图案纹身（条形码/靶心/狗爪印/锁链/烙印等）+ 繁中/日语侮辱性文字备选 + 四层皮肤融合系统 + 10 色墨水 + 位置锚定系统。

### 12 — 表情眼神 😍

## ⚙️ 不同模型对负面提示词的支持差异

负面提示词（Negative Prompt）并非所有 AI 绘画模型都支持：

| 模型/平台                             | 负面提示词支持 | 说明                    |
| ------------------------------------- | :------------: | ----------------------- |
| **ComfyUI**                     |       ✅       | 节点化，可精细控制权重  |
| **NovelAI**                     |       ✅       | 基于 SD 架构            |
| **Midjourney**                  |      ⚠️      | `--no` 参数，效果有限 |
| **Flux**                        |      ⚠️      | 取决于前端实现          |
| **DALL·E 3**                   |       ❌       | 无负面提示词机制        |
| **Z Image Turbo**               |       ❌       | 填了也会被忽略          |

### 💡 Z Image Turbo 用户注意

此模型的负面提示词不会生效。替代策略：

1. 正面描述中写否定词（效果有限）
2. 用精确解剖词替代泛词（`anatomically correct vulva` > `pussy`）
3. 用量词约束液体（`single drop` > 不约束）
4. 多次生成择优
