## 🎲 场景生成规则

> 📐 景别/视角/设备选择 → 见 [`07-风格胶片.md`](07-风格胶片.md)（完整词库 + 联动速查表 + 设备×画质对应）

**每次生成 prompt 时，从以下模块中随机组合：**

```
[场景 1个] + [主题/角色 1个] + [景别 1个] + [视角 1个] + [设备 1个] + [裸露/服装 1-2个] = 场景设定
```

**场景→设备联动速查**（详情见 07-风格胶片.md § 胶片×设备×场景联动速查）：

- **私密场景** → 手机 + POV/偷窥视角 + 近景
- **偷窥场景** → 监控/针孔 + 偷窥视角 + 远景/全景
- **AV封面** → 单反 + 平视 + 中景 + 高饱和胶片
- **复古场景** → 胶片相机 + Classic Negative/Gold

**⚠️ 注意：**

- 场景设定放在 prompt 最前面，引导整体画面氛围
- 裸露/服装紧跟拍摄设备之后，包含裸露程度和具体衣物状态
- 裸露程度要和场景/主题匹配（温泉→全裸，OL→制服半解，AV封面→裸露且有封面感）
- 每个 prompt 只能选 1 个场景 + 1 个主题 + 1 个景别 + 1 个视角 + 1 个设备 + 1-2 个裸露/服装

### 🚗 场景合理性铁律（写场景前必须过！）

**核心：先想"现实里做得到吗"，再写提示词。**

**三步检查：**
1. **物理可行性**：有支撑点吗？重心在哪？会摔倒吗？
2. **动作矛盾**：一个人能同时做两个需要专注的动作吗？
3. **空间关系**：两人的器官能接触到吗？
