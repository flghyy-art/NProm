## 📐 景别词库（Shot Size）

> 决定人物占画面的比例，直接影响亲密感和代入感。
> **用法**：每个 prompt 选 **1 个**景别。

### 标准景别（距离递增）

| 景别 | 英文关键词 | 说明 |
|------|-----------|------|
| **大特写 ECU** | `extreme close-up, macro detail shot` | 嘴唇/眼睛/局部，毛孔可见 |
| **近景 CU** | `close-up, face shot, face filling frame` | 面部充满画面，强调表情眼神 |
| **中近景 MCU** | `medium close-up, chest-up, upper body` | 胸部以上，突出上半身动作 |
| **中景 MS** | `medium shot, waist-up, waist to head` | 腰部以上，最常用对话景别 |
| **中全景 MLS** | `medium long shot, knee-up shot` | 膝盖以上，全身3/4展示 |
| **全景 LS** | `long shot, wide shot, full body shot` | 从头到脚，人物与环境平衡 |
| **远景 ELS** | `extreme long shot, extreme wide shot` | 人物极小，环境主导 |

## 📸 镜头焦段（Lens Focal Length）

> ⚠️ **使用规则**：仅专业相机/电影机使用。手机、监控、网络摄像头 **不加焦段**（固定镜头）。

| 焦段 | 关键词 | 视觉特征 | 适用景别 |
|------|--------|---------|---------|
| **超广角 14-24mm** | `ultra wide angle, 24mm lens, fisheye effect, wide distortion` | 夸张透视，边缘变形 | 全景/远景 |
| **广角 28-35mm** | `wide angle, 35mm lens, environmental context, natural perspective` | 环境感强，纪录片视角 | 全景/中全景 |
| **标准 50mm** | `50mm lens, standard focal length, natural field of view` | 接近人眼，不夸张不压缩 | 中景/中近景 |
| **中长焦 85mm** | `85mm lens, portrait lens, flattering compression, shallow depth of field` | 人像最佳，奶油焦外 | 近景/中近景 |
| **长焦 135mm+** | `135mm lens, telephoto, background compression, distant observer` | 远距离偷窥，压缩空间 | 特写/远景偷拍 |

## 🎲 生成规则

### 装配流水线

```
[场景] → [景别 1个] → [视角 1个] → [设备 1个] → [裸露/服装] → [外貌] → [姿势] → [表情] → [镜头焦段 0-1个] → [胶片风格 0-1组] → [构图 0-1个] → [风格标签 1个] → [画质强化]
```

### 联动规则

| 规则 | 说明 |
|------|------|
| **设备→焦段** | 手机/监控/网络摄像头 → 不加焦段（固定镜头） |
| **设备→画质** | 监控/CCTV → 不用 `masterpiece/8k`；胶片 → 不用 `8k/ultrahd` |
| **焦段→景别** | 85mm→近景/特写；50mm→中景；35mm→全景；135mm+→偷窥远景 |
| **视角→景别** | 鸟瞰→全景/远景；过肩→中景/近景；POV→近景 |
| **胶片→场景** | 暖调胶片→私密/情欲场景；冷调→医疗/科技；黑白→残酷/艺术 |
| **风格→设备** | 偷拍写实→手机/监控；AV封面→单反+影棚；复古→胶片相机 |

### 随机性规则

- **私密场景**：优先手机 + POV/偷窥视角 + 近景
- **正式拍摄**：优先单反/胶片 + 平视 + 中景/全景
- **偷窥场景**：优先监控/针孔 + 偷窥视角 + 远景/全景
- **AV封面**：优先单反 + 平视 + 中景 + Kodak Ektar/Velvia
- **复古场景**：优先胶片相机 + Classic Negative/Gold + 平视 + 中景

### ⚠️ 关键禁忌

| # | 禁忌 | 原因 |
|---|------|------|
| 1 | 监控/CCTV + `masterpiece, 8k` | 监控不可能8K画质 |
| 2 | 手机自拍 + `85mm lens` | 手机固定广角，无85mm |
| 3 | 胶片相机 + `8k, ultrahd` | 胶片没有数字分辨率 |
| 4 | 设备画质与风格矛盾 | 偷拍≠影棚质感 |
| 5 | 视角与景别不匹配 | 鸟瞰不适合面部特写 |
